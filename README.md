# KALIRA · by Cecy Lara

Sitio web estático (multi-página) para la marca de bienestar **Kalira**: regulación del sistema nervioso, biomagnetismo, longevidad y experiencias de bienestar. Estética "mármol y dorado", diseño fino y responsive.

## Contenido de la carpeta (todo en un solo nivel)

**Páginas (HTML):**
`index.html` (Inicio) · `filosofia.html` · `experiencias.html` · `cecy.html` · `contacto.html`

**Estilos y funciones:**
`styles.css` · `main.js`

**Imágenes:**
`marble.svg` (textura) y las fotos `.jpg`

Todo está suelto en la misma carpeta: no hay subcarpetas.

## Ver localmente

Abre `index.html` en cualquier navegador. Las páginas se enlazan entre sí por el menú.

## Subir a GitHub

Sube **todos los archivos de esta carpeta** (con `index.html` incluido) a tu repositorio.

## Publicar en GitHub Pages

1. En el repositorio: **Settings → Pages → Build and deployment → Source: Deploy from a branch → `main` / `/ (root)`**.
2. En unos minutos tu sitio estará en `https://<tu-usuario>.github.io/<tu-repo>/`.

El archivo `.nojekyll` ya está incluido para que GitHub Pages sirva todos los archivos correctamente.
